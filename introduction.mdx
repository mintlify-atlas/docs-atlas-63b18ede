    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get your pharmacy management system running in minutes
  </p>

  <Steps>
    <Step title="Download and extract">
      Clone the repository or download the source code to your Apache web server directory.
      
      ```bash
      git clone https://github.com/gfrodriguez/inventario.git
      cd inventario
      ```
    </Step>

    <Step title="Configure database">
      Import the database schema and configure your connection settings.
      
      ```bash
      mysql -u root -p < db/database.sql
      ```
      
      Edit `db/db.php` with your MySQL credentials:
      
      ```php
      $_user = 'your_username';
      $_pass = 'your_password';
      $db = new PDO('mysql:host=localhost;dbname=drogueria', $_user, $_pass);
      ```
    </Step>

    <Step title="Enable mod_rewrite">
      The system requires Apache's mod_rewrite module for friendly URLs. Enable it if not already active:
      
      ```bash
      sudo a2enmod rewrite
      sudo systemctl restart apache2
      ```
    </Step>

    <Step title="Log in and start managing">
      Navigate to your installation URL and log in with the default admin account:
      
      - **Username:** admin
      - **Password:** 123456
      
      <Warning>
        Change the default password immediately after your first login for security.
      </Warning>
    </Step>
  </Steps>
</div>
